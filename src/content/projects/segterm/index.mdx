Segterm is an experimental terminal emulator that explores low-level framebuffer rendering and VT100/ANSI escape sequence handling.

## Features

- Framebuffer-based rendering
- VT escape sequence parsing
- Minimal footprint

## Tech

Rust, Linux framebuffer
