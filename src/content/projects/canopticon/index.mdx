Canopticon analyzes photos to estimate the percentage of sky visible through a forest canopy. It uses image segmentation to identify sky pixels and compute a cover ratio.

## Features

- Photo-based canopy analysis
- Sky segmentation pipeline
- Outputs percentage and annotated image

## Tech

Python, OpenCV, Raspberry Pi
